= NetApp GenAI 엔진의 구성 요소
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
GenAI 인프라를 배포하면 Workload Factory가 GenAI 엔진에 대한 EC2 인스턴스를 생성합니다.  또한 이 인스턴스에 대한 IAM 역할, 보안 그룹 및 개인 엔드포인트가 생성됩니다.  AWS 환경에서 Workload Factory가 생성하는 이러한 구성 요소에 대해 더 자세히 알아보고 싶을 수도 있습니다.

EC2 인스턴스 유형입니다:: m5.large를 참조하십시오
IAM 역할:: GenAI 엔진 인스턴스에서는 데이터 집합을 Amazon Bedrock의 내장 모델에 전송하고 NetApp AI Service 백엔드와 통신할 수 있는 권한이 필요합니다. IAM 역할에는 다음 권한이 포함됩니다.
+
